To view the "Mack Address" of the machine follow the steps:-

Go to gem file and copy this gem name.
1. gem 'macaddr', '~> 1.7.1'

Install bundle in tha app.
2. Bundle

Access in related folder (controller)
3. require 'macaddr'

To get "Mac Address"   
4. Mac.addr 

and We will have the mack address in a formate as: 

60:60:60:60:1c:d1
